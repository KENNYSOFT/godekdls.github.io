이번 레퍼런스 문서에선 데이터 접근과 관련된 이야기를 주로 다루며, 데이터 접근 레이어와 비즈니스(서비스) 레이어 간 상호 작용에 관해 설명한다.

스프링의 트랜잭션 관리 기능을 종합적으로 설명하며, 스프링 프레임워크와 통합되는 다양한 데이터 접근 프레임워크와 기술을 함께 다룬다.
