---

이 프로젝트를 사용하면 스프링 5, 스프링 부트 2, 프로젝트 리액터를 비롯한 스프링 에코시스템 위에 API 게이트웨이를 구축할 수 있다. 스프링 클라우드 게이트웨이의 주목적은 간단하면서도 확실한 방법으로 요청을 다른 API로 라우팅하고, 보안, 모니터링/메트릭, resiliency같은 횡단 관심사를 모아 처리하는 거다.
